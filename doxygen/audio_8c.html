void audio_play_notes ( char *  notes)

Play a string of characters

All notes are 1/8 notes. However if you add a cipher before a note the length will be multiplied with it. For 1/4 notes write '2A', for 1/2 notes write 4A and for 1/1 notes write '8A'. If you need a higher octave you can write 'x' before a note. For example write xA for A4, and just A for A3. Supported notes are A3 to G5.

Parameters
notesThe notes to play; Supports only uppercase characters, numbers and 'x'; Must end with '\0'
